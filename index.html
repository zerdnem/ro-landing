Installation

  • Download the latest kRO client. This is the main Ragnarok Online file and you can download it HERE
  • Next is to download our client. These are files that we need inorder to connect to the server. LINK HERE, ANOTHER LINK
  • After download, just click the installer Full_kRO_Renewal_20150601_.exe then install it.
  • Extract client.7z then copy the files inside it to the main RO folder except for the System folder.
  • Then copy the files inside the System folder to the main RO/System folder.
  • Now click 2013-08-07aRagexe_patched.exe to run the game but first you must register a new account. Read the FAQ for more info.

FAQ

How to register?

  • Visit our control panel HERE. This is where you manage your account and whatnot.

How to update the client?

  • Run rsu-kro-rag-lite.exe and rsu-kro-renewal-lite.exe inside the main RO folder. Just don't click start after the update.

How to change resolution, disable sound, etc.?

  • Just run opensetup.exe.

What RO emulator are you using?

  • I'm using rAthena instead of Hercules. Here is a good discussion on which is a better emulator.

What is the exp and drop rates?

  • You can type @rates ingame to see the rates.

What are other commands that I can use ingame?

  • You can use commands like @save @go @autoloot @storage @rates @check @identify.

Wow, your FAQ suck!

  • Just make a pull request then change things that you don't want to see.
